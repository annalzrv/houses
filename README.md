# houses

Investment and leasing analysis for two Brooklyn buildings held by the same ownership group:

- 61 Greene Avenue (Fort Greene / Clinton Hill)
- 301 Washington Avenue (Clinton Hill)

The goal is to find the most profitable legal way to configure and lease each building, with the owner living in one unit, and to give the building manager a clear set of actions.

## Method

Property facts come from NYC public records (Department of Finance, PLUTO, DOB, GeoSearch). Rents come from current furnished listings in the relevant neighborhoods, cross-checked against local rent indices. Property taxes are verified figures; other operating expenses are estimates flagged as such and meant to be replaced with the buildings' actual bills. Financing is excluded from the pro formas.
